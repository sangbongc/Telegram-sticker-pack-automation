# Telegram-sticker-pack-automation
Python automation tool for creating Telegram sticker packs from local media files.

## Overview

This project automates the creation of Telegram sticker packs using Python.

It converts images, GIFs, and videos into Telegram-compatible WebM stickers and uploads them automatically via the Telegram Bot API.

## Features

- Convert PNG/JPG/GIF/MP4 to WebM
- Resize media for Telegram sticker requirements
- Asynchronous batch processing
- Automatic sticker pack creation
- Timestamp-based output management

## Technologies

- Python
- Telegram Bot API
- FFmpeg
- asyncio
- tkinter
